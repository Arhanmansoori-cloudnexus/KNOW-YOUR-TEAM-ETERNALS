# Team Eternals CN - Employee Directory

A modern, responsive employee directory application specifically designed to showcase the members of **Team Eternals CN**. This tool provides a centralized hub for team information, technical expertise, and professional profiles.

## 🌟 Features

- **Complete Team Directory**: Full listing of all Team Eternals members.
- **Individual Profile Pages**: Dedicated space for every employee’s detailed professional background.
- **Advanced Search & Filter**: Instant lookup by name, designation, department, or location.
- **Modern UI/UX**: A sleek interface featuring Glass-morphism, gradient effects, and smooth animations.
- **Responsive Design**: Optimized for seamless viewing on Desktop, Tablet, and Mobile.
- **Bilingual Support**: User-friendly interface supporting both English and Hindi.

## 🚀 Technology Stack

- **Frontend**: React.js 18 with Vite
- **Routing**: React Router DOM
- **Styling**: Custom CSS with Modern Design Patterns (Glass-morphism)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 📋 Employee Information Included

Each profile within the directory captures the following data points:

- 📸 **Identity**: Employee photo/avatar and full name.
- 💼 **Role**: Job title, department, and official location.
- 📞 **Connectivity**: Official contact details and Time Zone/Shift.
- 📅 **Tenure**: Date of joining.
- 🛠️ **Expertise**: Key professional skills and specialized technical areas.
- 📝 **Biography**: Professional introduction and career summary.
